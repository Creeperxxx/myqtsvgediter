## **4. PowerPoint (.pptx) 文档结构详解**
- 各关键组成部分的作用说明：
  - `ppt/presentation.xml`（演示文稿整体结构）
  - `ppt/slides/`（每张幻灯片）
  - `ppt/slideMasters/` 和 `slideLayouts/`（母版与布局）
  - `ppt/media/`（多媒体资源）
